### Overview

Uzhavar AI is a comprehensive Streamlit-based web application designed to assist farmers with intelligent agricultural tools. The app provides bilingual support (English and Tamil) and features multiple modules for crop recommendation, weather updates, fertilizer guidance, and historical data analysis.

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```

### Usage

1. Select your preferred language (English/Tamil)
2. Navigate through the sidebar to access different features
3. Input your farming parameters for personalized recommendations
4. View historical data and analytics in the dashboard

### Data
